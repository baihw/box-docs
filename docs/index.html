简介

一个希望能够保持简约，并且具有较强的环境适应性的快速开发框架。

目标

  • 简化开发模型，降低心智负担
  • 抽象依赖，标准化，敏捷和可替换性
  • 不因技术潮流的变更影响到业务代码的复用性
  • 对海量生态组件的价值筛选、兼容和集成保证以及支撑服务
